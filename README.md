# Crewly

A boat calendar app for organizing days on the water. Manage your boats, invite crew, track RSVPs, and check the weather — all in one place.

## Features

- **Boat & crew management** — own multiple boats, invite crew by email, manage member roles
- **Drag-and-drop scheduling** — plan sessions on a monthly calendar, drag crew onto dates
- **RSVP tracking** — crew can confirm, decline, or stay pending with color-coded status indicators
- **10-day weather forecasts** — Google Weather API integration with location-based forecasts and calendar styling that adapts to conditions
- **Contacts** — per-boat contact lists with full CRUD
- **Dark mode** — light/dark theme toggle

## Tech stack

React 19 · TypeScript · Vite · Tailwind CSS · Convex (backend + auth + real-time sync)
